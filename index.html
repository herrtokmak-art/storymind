✍️ Response Format Rules

*asterisks* → bold (narration/action), _underscores_ → italics (thoughts), "quotes" → spoken dialogue.
🧠 Memory
🛠️ Memory builder prompt (every 10 messages)
Produces a structured JSON memory. Use {{char}} and {{user}}.
🔍 Query generator prompt (each message)
Turns the current moment into a search query over stored memories.
📊 Memory ranker prompt (each message)
Picks the most relevant memories. Use {{maxCount}}.
🗜️ Memory condenser prompt (distant memories)
Merges old low-importance memories into durable summaries. Use {{char}} and {{user}}.
🪪 Character
🪪 Character bio generator prompt
Turns a brief (or conversation context) into a full character card as JSON. Used by "Create with AI" and auto-character.
🌌 Universe generator prompt
Turns a brief + character count into a full universe (setting, your background, and a cast) as JSON.
📊 Story Systems
❤️ Relationship tracker prompt
Decides how feelings shift each turn. Returns JSON deltas.
📊 Freeform tracker prompt
Judges soft deltas and detects trigger phrases. Dice/day-counting handled by the app.
⚠️ Voice consistency prompt
Flags out-of-character replies. Returns {ok, note}.
🗺️ Travel narration prompt
Narrates the journey from one place to another. Use {{user}}.
📅 Calendar plan-detector prompt
Extracts future plans from conversation as JSON.
🎬 Gamemaster
⚖️ Staleness judge prompt (stage 1)
Returns JSON {stale, reason, energy}. Use {{user}} and {{sensitivity}}.
✍️ Event author prompt (stage 2)
Writes the hidden nudge. Use {{user}} and {{reason}}.
🌙 Day-summary prompt (End Day button)
Condenses a finished day into one memory. Use {{char}} and {{user}}.
🎬 Scene Writer
🧩 Event setup prompt
Classifies a new event. Returns JSON.
✍️ Scene advance prompt
Returns JSON. Use {{user}}, {{summary}}, {{turn}}, {{min}}, {{max}}.
🎭 Multi-Character Director
🗣️ Router 1 — after the player speaks
Returns JSON {addressed, responders}. Use {{user}}.
↔️ Router 2 — after a character speaks
Returns JSON {continue, responder, addressed}. Use {{user}} and {{speaker}}.
🚶 Presence tracker prompt
Detects entrances/exits. Returns JSON {exit, enter}. Use {{user}}.

New Universe

📊 Trackers
Named values the story tracks (love, trust, pregnancy…). Define them here per universe; each new chat starts fresh. The AI nudges them as the scene unfolds; stage text is fed back into the story.
🗺️ Locations
Places in this world — homes where characters live and points of interest to travel to. You can generate a picture for each; the chat background changes when you're there.

Who's nearby?

Every character in this universe is part of the cast. Only those marked 🟢 nearby hear you and can respond. Bring someone nearby or send them away here, or use OOC commands in chat: /bring Name, /leave Name, /who. With nobody nearby, you're narrating alone until the Gamemaster or you bring someone in.
